# Packing and Pre-Departure Checklist

## Before Booking

- [ ] Confirm trip dates.
- [ ] Confirm arrival and departure airport.
- [ ] Decide regions: Panama City, Canal/nature, Boquete, Bocas del Toro, San Blas/Guna Yala, Pacific coast, other.
- [ ] Confirm travel companions and room needs.
- [ ] Decide budget range and comfort level.
- [ ] Check passport expiration date.

## 4-8 Weeks Before

- [ ] Book flights.
- [ ] Book lodging.
- [ ] Decide whether the itinerary includes malaria-risk areas.
- [ ] Schedule travel clinic or primary care visit if needed.
- [ ] Review CDC Panama page.
- [ ] Confirm travel insurance and medical evacuation coverage needs.
- [ ] Check phone roaming/eSIM plan.
- [ ] Make copies of passport and key reservations.

## 1-2 Weeks Before

- [ ] Download offline maps.
- [ ] Save lodging addresses in phone notes.
- [ ] Save emergency contacts offline.
- [ ] Confirm airport transfer or first-night transport.
- [ ] Check baggage limits for any domestic flights.
- [ ] Notify card issuer if needed.
- [ ] Pull small US bills.
- [ ] Recheck weather for each region.

## Documents

- [ ] Passport.
- [ ] Return or onward ticket.
- [ ] Lodging confirmations.
- [ ] Travel insurance card/details.
- [ ] Emergency contacts.
- [ ] Prescription list with generic names.
- [ ] Driver's license and International Driving Permit if renting a car.

## Clothes

- [ ] Lightweight shirts.
- [ ] Lightweight pants/shorts.
- [ ] Swimwear.
- [ ] Light rain jacket.
- [ ] Comfortable walking shoes.
- [ ] Sandals/water-friendly footwear.
- [ ] Hat.
- [ ] Sunglasses.
- [ ] One warmer layer for highlands, buses, and air conditioning.

## Health and Toiletries

- [ ] Sunscreen.
- [ ] Insect repellent.
- [ ] Personal prescriptions plus extra.
- [ ] Anti-diarrheal.
- [ ] Oral rehydration salts.
- [ ] Pain reliever/fever reducer.
- [ ] Basic first aid/blister kit.
- [ ] Hand sanitizer.
- [ ] Motion-sickness treatment if taking boats or mountain roads.

## Gear

- [ ] Daypack.
- [ ] Dry bag or zip bags.
- [ ] Charger and power bank.
- [ ] Reusable water bottle.
- [ ] Small flashlight/headlamp if staying in rustic areas.
- [ ] Earplugs/eye mask.
- [ ] Lightweight lock if staying in hostels/shared spaces.

## Money

- [ ] Primary credit/debit card.
- [ ] Backup card stored separately.
- [ ] Small US bills.
- [ ] Emergency cash reserve.

## Final Day

- [ ] Passport in personal item.
- [ ] Wallet/phone/charger in personal item.
- [ ] First-night lodging address accessible offline.
- [ ] Airport transfer plan confirmed.
- [ ] Key documents shared with a trusted person.
